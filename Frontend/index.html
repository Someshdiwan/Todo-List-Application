Todo List Application

This page hosts the runnable JAR for the console application. Download and run locally with Java 17+.

Download app.jar

Run locally

java -jar app.jar

How it works

  1. The repository builds a runnable JAR via GitHub Actions.
  2. Actions publishes a `gh-pages` branch containing this page + app.jar.
  3. Open the page URL and click Download app.jar. Run the jar on your machine.

If you need the app to run in the browser, we can convert the code to a web API or use TeaVM to compile to JavaScript — but that requires code changes.
